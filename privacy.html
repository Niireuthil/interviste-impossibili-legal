Privacy Policy — Interviste Impossibili Scheduler

Last updated: 2026-06-16

This is an internal tool built and operated by the creator of the "Interviste Impossibili" project to automate publishing of already-approved video content to the project's own social media accounts (Instagram, Facebook, TikTok).

What this app does

The app uses the TikTok Content Posting API (and equivalent APIs on other platforms) solely to publish video content — created and approved in advance by the project owner — to the project's own TikTok account, on a schedule.

Data we access

The only TikTok account data accessed is the OAuth access token issued by TikTok to authorize posting on behalf of the project's own TikTok account. We do not access, collect, or store any data belonging to other TikTok users, viewers, or third parties.

Data storage

The access token is stored as an encrypted secret inside GitHub Actions (a private CI/CD environment) and is never exposed publicly or shared with third parties. Video files are stored temporarily in a private Cloudflare R2 bucket solely so they can be fetched by TikTok at publish time, and are deleted afterwards.

Data sharing

We do not sell, share, or otherwise disclose any data to third parties.
